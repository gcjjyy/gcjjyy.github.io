    하이텔 게제동(GMA) 공개자료실 백업사이트

    게임 개발자연대 오영욱 이사님께서, 파란 서비스가 종료하기 전에 게제동(게임제작동호회/GMA) 자료실을 백업받아놓으셨었다.

    해당 자료를 전달받아, Firebase Hosting / Firestore + React.js + Chakra UI, Algolia를 이용해서 이전 자료들을 웹상에서 쉽게 검색하고 다운받을 수 있게 만들었다.

    Firestore자체에서 Text 검색 기능을 지원하지 않고, 공식 설명에도 Algolia같은 3rd Party 검색서비스를 사용하라고 나와있다. 설명대로 Algolia서비스를 일단 무료로 사용하고 있고, 하이텔 게제동 자료에 관심있는 사람들이 그리 많지 않기 때문에 무료로도 충분히 사용이 가능하다.

    자료 업로드는 node.js를 이용해 자동 업로드 툴을 만들어서 쉽게 업로드 하였다.

    이 사이트는 최초에 Vue.js + Vuetify로 만들었다가, React + Bootstrap으로 만들었다가, React + Chakra UI로 만들었다. 동일한 기능을 하는 사이트를 3번 거의 새로 만들었다.

    좋은 공부가 되었다.
