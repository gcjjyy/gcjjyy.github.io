    Fedora 38 Flutter 설치

    Fedora Linux 38에 Flutter 개발환경 설치 (Command line)

    Flutter 다운로드 및 압축 해제 (원하는 디렉토리에)

    cd ~/Downloads
wget https://storage.googleapis.com/flutter_infra_release/releases/stable/linux/flutter_linux_3.10.2-stable.tar.xz
cd ~
tar xvf ~/Downloads/flutter_linux_3.10.2-stable.tar.xz

    압축 해제 후 flutter의 bin디렉토리를 PATH에 걸어주어야 좋다.

    기본적으로 필요한 구성요소 설치

    flutter doctor 명령어를 쳐가면서 설치해도 큰 문제가 없으나, 여기에 정리해 둔다.

    sudo dnf install clang
sudo dnf install cmake ninja-build
sudo dnf install gtk3-devel

    안드로이드 스튜디오 및 command line tools 설치 (다운로드 및 압축해제)

    cd ~/Downloads
wget https://redirector.gvt1.com/edgedl/android/studio/ide-zips/2022.2.1.20/android-studio-2022.2.1.20-linux.tar.gz
wget https://dl.google.com/android/repository/commandlinetools-linux-9477386_latest.zip
cd ~
tar xvf ~/Downloads/android-studio-2022.2.1.20-linux.tar.gz
unzip ~/Downloads/commandlinetools-linux-9477386_latest.zip

    commandlinetools 의 PATH도 걸어주는 것이 좋다.

    안드로이드 SDK 설치 (33버전 기준)

    sdkmanager --sdk_root=/home/gcjjyy/android-studio/sdk "platform-tools" "platforms;android-33"
sdkmanager --sdk_root=/home/gcjjyy/android-studio/sdk "cmdline-tools;latest"
sdkmanager --sdk_root=/home/gcjjyy/android-studio/sdk "build-tools;33.0.2"

    Java 설치

    sudo dnf install java-11-openjdk.x86_64
cd /usr/lib/jvm/java-17-openjdk-17.0.7.0.7-5.fc38.x86_64/
mkdir ~/android-studio/jre
sudo cp -r * /home/gcjjyy/android-studio/jre/
cd ~/android-studio/jfe
sudo chown -R gcjjyy:gcjjyy .

    자바 설치부분이 독특한데, JDK를 설치한 후 설치된 경로의 모든 파일들을 android-studio가 설치된 디렉토리에 jre라는 이름으로 복사해주어야 한다. 이것은 Flutter가 JRE를 찾는 방식과 Android studio for Linux의 차이에서 오는 문제 같다. 자세한 내용은 구글링하면 많이 나온다.

    SDK Path 설정 및 라이센스 동의

    flutter config --android-sdk=/home/gcjjyy/android-studio/sdk
flutter doctor --android-licenses

    Chrome 설치

    sudo dnf install fedora-workstation-repositories
sudo dnf config-manager --set-enabled google-chrome
